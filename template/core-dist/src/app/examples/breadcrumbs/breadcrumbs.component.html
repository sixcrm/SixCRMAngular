Breadcrumbs

breadcrumbs are a good way to display your current location. This is usually used when you have multiple layers of content.

Breadcrumbs are auto-generated from the menu items hierarchy and can be overridden using the NavigationService.setBreadcrumbs([{{'{'}}title : 'Examples', link : ''}, {{'{'}}title : 'Multiple', link : ''}, {{'{'}}title : 'Levels', link : ''}]);

Page Title is also auto-generated and can also be overridden using the NavigationService.setPageTitle('My page title');

Browser Title is auto-generated from the 'App Name', 'Title Separator', and 'Page Title' combined. Like the others, it can be overridden using NavigationService.setBrowserTitle('Testing Title'); or you can modify each part of the auto-generated title as well (i.e. NavigationService.setTitleSeparator('-');).
